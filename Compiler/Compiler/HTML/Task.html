Тема: Объявление списка с инициализацией на языке Python

Выполнил: Бикмухаметов Таир Олегович, группа АВТ-314, НГТУ

Описание

Список — это упорядоченная последовательность элементов. В данной работе элементом является константа.

Константой может быть:

  • целое число без знака;
  • вещественное число без знака;
  • целое число со знаком;
  • вещественное число со знаком;
  • строка в одинарных кавычках;
  • булевы значения True / False.

Формат записи:

имя_списка = [элементы_списка];

Примеры корректных строк

a = [];
a = ['Sun'];
a = [1, -2, +3.4, True, False, 'str'];
